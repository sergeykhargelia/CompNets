## Задание 1
Запуск сервера: 
```shell
python3 server.py <max_connections>
```

Здесь max_connections это максимальное количество запросов, которые серверу разрешено обрабатывать одновременно. 

Запуск клиента: 
```shell
python3 client.py <host> <port> <path_to_file>
```
